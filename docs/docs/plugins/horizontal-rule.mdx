The Horizontal Rule plugin allows you to easily insert a horizontal line, or horizontal rule, within the Plate editor.

This visual element can be used to separate content, indicate a shift in topic between paragraphs or sections, or enhance the overall layout of the content.

Use this plugin to render a `<hr>` HTML tag.

Type three dashes (`---`) or three underscores and a space (`___ `) at the beginning of a new line and it will magically transform to a horizontal rule.

### Demo

To add a horizontal rule in the provided demo, type three dashes (**`---`**) or three underscores followed by a space (**`___`** ) at the beginning of a new line, and it will automatically transform into a horizontal line.

### Installation

Follow these steps to integrate the horizontal rule plugin into your Plate editor:

1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-horizontal-rule
npm install @udecode/plate-ui-horizontal-rule
npm install @udecode/plate-autoformat
```

2. Import the plugin and include it in your plugin list:

```
import { createAutoformatPlugin } from '@udecode/plate-autoformat';
import { createHorizontalRulePlugin } from '@udecode/plate-horizontal-rule';
import { createSelectOnBackspacePlugin } from '@udecode/plate-select-on-backspace';
import { basicNodesPlugins, ELEMENT_DEFAULT, ELEMENT_HR } from '@udecode/plate';
import { setNodes, insertNodes } from '@udecode/plate-common';
import type { AutoformatPlugin } from '@udecode/plate-autoformat';
import type { MyEditor, MyValue } from '../types';

const plugins = createPlugins([
  ...otherPlugins,
  ...basicNodesPlugins,
  createHorizontalRulePlugin(),
  createSelectOnBackspacePlugin({
    options: { query: { allow: [ELEMENT_HR] } },
  }),
  createAutoformatPlugin<AutoformatPlugin<MyValue, MyEditor>, MyValue>({
    options: {
      rules: [
        {
          mode: 'block',
          type: ELEMENT_HR,
          match: ['---', 'â€”-', '___ '],
          preFormat,
          format: (editor) => {
            setNodes(editor, { type: ELEMENT_HR });
            insertNodes(editor, {
              type: ELEMENT_DEFAULT,
              children: [{ text: '' }],
            });
          },
        },
      ],
    },
  }),
], {
  components: plateUI,
});

```
