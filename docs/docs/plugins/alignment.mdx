The Alignment plugin provides support for text alignment, enabling you to align content to the left, right, center, or justify it.
This feature enhances the visual presentation of your text.

### Demo

To apply the desired alignment to a paragraph, click within the paragraph or header, and then use the Text Alignment option on the toolbar dropdown menu.

Click inside a paragraph or a header and click on a toolbar button to apply the desired text alignment.

### Installation

Follow these steps to integrate the alignment plugin into your Plate editor:


1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-alignment
npm install @udecode/plate-ui-alignment
```

2. Import the plugin and add it to your plugin list:

```
import { createAlignPlugin } from '@udecode/plate-align';
import { basicNodesPlugins } from '@udecode/plate-basic-nodes';
import {
  ELEMENT_H1,
  ELEMENT_H2,
  ELEMENT_H3,
  ELEMENT_H4,
  ELEMENT_H5,
  ELEMENT_H6,
  ELEMENT_PARAGRAPH,
} from '@udecode/plate';

const plugins = createPlugins([
  ...otherPlugins,
  ...basicNodesPlugins,
  createAlignPlugin({
    inject: {
      props: {
        validTypes: [
          ELEMENT_PARAGRAPH,
          ELEMENT_H1,
          ELEMENT_H2,
          ELEMENT_H3,
          ELEMENT_H4,
          ELEMENT_H5,
          ELEMENT_H6,
        ],
      },
    },
  }),
], {
  components: plateUI,
});

```
