The Single Line plugin limits the editor to only one line of text, making it suitable for certain types of content,
such as headlines or short phrases, where a single line is sufficient.

### Demo

Test out the Single Line feature in the demo provided.

It limits the editor to one line of text, perfect for brief content or titles.










import { SingleLineSandpack } from "./SingleLineSandpack";

<SingleLineSandpack />

### Installation

Follow these steps to integrate the single line plugin into your Plate editor:


1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-break
```

2. Import the plugin and add it to your plugin list:

```
import { createListPlugin } from '@udecode/plate-list';
import { createSingleLinePlugin } from '@udecode/plate-single-line';
import { createTablePlugin } from '@udecode/plate-table';

const plugins = createPlugins([
...otherPlugins,
...basicNodesPlugins,
createListPlugin(),
createTablePlugin(),
createSingleLinePlugin(),
], {
components: createPlateUI(),
});

```
