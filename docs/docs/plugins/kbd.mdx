Plate comes with a set of preconfigured keyboard shortcuts to enhance productivity and ease of use.

While these defaults cater to most user needs, you may want to modify the shortcuts to better suit your specific requirements.

### Demo

you can use the following list of keyboard shortcuts to get started:

- Bold: **`Ctrl/Cmd + B`**
- Italic: **`Ctrl/Cmd + I`**
- Underline: **`Ctrl/Cmd + U`**
- Strikethrough: **`Ctrl/Cmd + Shift + X`**
- Code: **`Ctrl/Cmd + Shift + C`**
- Blockquote: **`Ctrl/Cmd + Shift + Q`**
- Code Block: **`Ctrl/Cmd + Shift + K`**
- Heading 1-6: **`Ctrl/Cmd + 1-6`**
- Paragraph: **`Ctrl/Cmd + 0`**
- Undo: **`Ctrl/Cmd + Z`**
- Redo: **`Ctrl/Cmd + Shift + Z`**
- Cut: **`Ctrl/Cmd + X`**
- Copy: **`Ctrl/Cmd + C`**
- Paste: **`Ctrl/Cmd + V`**
- Select All: **`Ctrl/Cmd + A`**

### Installation

Follow these steps to integrate the keyboard shortcuts plugin into your Plate editor:


1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-kbd
```

2. Import the plugin and add it to your plugin list:

```
import { createKbdPlugin } from '@udecode/plate-kbd';

const plugins = createPlugins([
  ...otherPlugins,
  ...basicNodesPlugins,
  createKbdPlugin(),
], {
  components: createPlateUI(),
});

```
