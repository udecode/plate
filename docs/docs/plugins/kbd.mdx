### Installation

Follow these steps to integrate the keyboard input plugin into your Plate editor:

1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-kbd
```

2. Import the plugin and add it to your plugin list:

```tsx
import { createKbdPlugin } from '@udecode/plate';

const plugins = createPlugins([
  // ...otherPlugins,
  createKbdPlugin(),
], {
  components: createPlateUI(),
});
```
