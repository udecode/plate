The Highlight plugin provides text marking tools to aid content authors in their work, such as reviewing and referencing content.

It utilizes inline **`<mark>`** elements, supporting markers (background color).

[//]: # (It utilizes inline **`<mark>`** elements, supporting both markers &#40;background color&#41; and pens &#40;text color&#41;, and comes with a flexible configuration.)

### Demo

To utilize the highlight feature, first select the desired text and then use the Highlight toolbar button to apply the highlight.

[//]: # (You can choose from a predefined and limited number of available colors.)

### Installation

Follow these steps to integrate the highlight plugin into your Plate editor:

1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-highlight
```

2. Import the plugin and add it to your plugin list:

```tsx
import { createHighlightPlugin } from '@udecode/plate-highlight';

const plugins = createPlugins([
  // ...otherPlugins,
  createHighlightPlugin(),
], {
  components: createPlateUI(),
});
```
