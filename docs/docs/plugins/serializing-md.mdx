The Deserialize Markdown plugin enables copying and pasting Markdown content from external sources like **[https://markdown-it.github.io/](https://markdown-it.github.io/)**.

This plugin simplifies the process of importing Markdown content into your Plate editor, ensuring seamless integration of formatted text.


### Demo

To see the Deserialize Markdown plugin in action, copy and paste Markdown content from **[https://markdown-it.github.io/](https://markdown-it.github.io/)** into your Plate editor.

The plugin will automatically convert the Markdown formatting to match the editor's styling.

### Installation

Follow these steps to integrate the deserialize markdown plugin into your Plate editor:

1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-serializer-md
```

1. Import the plugin and add it to your plugin list:

```
import React from 'react';
import { createDeserializeMdPlugin } from '@udecode/plate';
import { basicNodesPlugins } from './basic-nodes/basicNodesPlugins';
import { plateUI } from './common/plateUI';
import { deserializeMdValue } from './serializing-md/deserializeMdValue';
import { createMyPlugins, MyValue } from './typescript/plateTypes';

const plugins = createMyPlugins(
  [
    ...otherPlugins,
    ...basicNodesPlugins,
    createDeserializeMdPlugin(),
  ],
  {
    components: plateUI,
  }
);
```
