The line height plugin enables support for line height.
You can use it to set the line height for the selected element as required.

### Installation

```bash npm2yarn
npm install @udecode/plate-line-height
npm install @udecode/plate-line-height-ui
```

### Usage

Click inside a paragraph or a header and click on a toolbar button to apply the desired line height.

```ts live
() => {
  const plugins = [
    ...PLUGINS.basicNodes,
    createLineHeightPlugin({
      defaultNodeValue: 1.5,
      validNodeValues: [1, 1.2, 1.5, 2, 3],
      validTypes: [
        ELEMENT_PARAGRAPH,
        ELEMENT_H1,
      ],
    })
  ];

  return (
    <>
      <HeadingToolbar>
        <ToolbarLineHeight icon={<LineWeight />} />
      </HeadingToolbar>
      <Plate
        id="alignment"
        plugins={plugins}
        components={CONFIG.components}
        options={CONFIG.options}
        editableProps={CONFIG.editableProps}
        initialValue={VALUES.lineHeight}
      />
    </>
  );
}
```

### Options

- See [getOverrideProps](plugin-utils#getoverrideprops)
- Default options:

```ts
{
  nodeKey: KEY_LINE_HEIGHT,
  defaultNodeValue: DEFAULT_LINE_HEIGHT,
  validTypes: [getPlatePluginType(editor, ELEMENT_DEFAULT)],
  validNodeValues: DEFAULT_LINE_HEIGHTS,
}
```
