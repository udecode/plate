The Soft Break plugin allows users to insert line breaks within a block of text without starting a new block.

This can be useful for creating spacing or separating elements within a block of text without disrupting the structure of the document.

### Demo

To test the Soft Break feature, use the Shift+Enter keyboard shortcut in the provided demo.

The results will vary depending on the type of block you're currently working with.

### Installation

Follow these steps to integrate the soft break plugin into your Plate editor:

1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-break
```

2. Import the plugin and add it to your plugin list:

```tsx
import { createSoftBreakPlugin, ELEMENT_CODE_BLOCK, ELEMENT_BLOCKQUOTE, ELEMENT_TD } from '@udecode/plate';

const plugins = createPlugins([
  // ...otherPlugins,
  createSoftBreakPlugin({
    options: {
      rules: [
        { hotkey: 'shift+enter' },
        {
          hotkey: 'enter',
          query: {
            allow: [ELEMENT_CODE_BLOCK, ELEMENT_BLOCKQUOTE, ELEMENT_TD],
          },
        },
      ],
    },
  }),
], {
  components: createPlateUI(),
});
```

### Options

```tsx
export interface SoftBreakRule {
  hotkey: string;

  /**
   * Filter the block types where the rule applies.
   */
  query?: QueryNodeOptions;
}

export interface SoftBreakPlugin {
  rules?: SoftBreakRule[];
}
```
