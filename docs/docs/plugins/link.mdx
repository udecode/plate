### `createLinkPlugin`

A link is an inline element:
- default `type` is `'a'`
- `url` is the url of the link.

### Installation

```bash npm2yarn
npm install @udecode/plate-link
npm install @udecode/plate-link-ui
```

### Usage

```ts live
() => {
  const plugins = createPlugins([
    ...PLUGINS.basicNodes,
    createLinkPlugin()
  ], {
    components: CONFIG.components,
  });

  return (
    <>
      <HeadingToolbar>
        <LinkToolbarButton icon={<Link />} />
      </HeadingToolbar>
      <Plate
        id="link"
        plugins={plugins}
        editableProps={CONFIG.editableProps}
        initialValue={VALUES.link}
      />
    </>
  );
}
```
