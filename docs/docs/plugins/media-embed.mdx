Media providers supported by this plugin:

- video: 
  - youtube
  - vimeo
  - dailymotion
  - youku
  - coub
- twitter

### Installation

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-media-embed
npm install @udecode/plate-ui-media-embed
```

### Usage

import { MediaEmbedSandpack } from './MediaEmbedSandpack'

<MediaEmbedSandpack />
