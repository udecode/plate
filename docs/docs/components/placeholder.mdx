The Placeholder plugin for the RTE allows for the display of a customizable placeholder text when the editor is empty.
It helps users locate the editor in the application and prompts them to input content.
The feature works similarly to the native DOM Placeholder attribute used by inputs.
It can be configured to display a specific message or prompt for specific types of content.

### Demo

To test the Placeholder plugin, simply focus on an empty paragraph, and the placeholder will be shown.

Try to focus an empty paragraph: a placeholder will be shown.

### Installation

Follow these steps to integrate the placeholder plugin into your Plate editor:


1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-ui-placeholder
```

1. Import the plugin and add it to your plugin list:

```
import { createExitBreakPlugin } from '@udecode/plate-exit-break';
import { basicNodesPlugins } from '@udecode/plate-basic-nodes';

const plugins = createPlugins([
  ...otherPlugins,
  ...basicNodesPlugins,
  createExitBreakPlugin(exitBreakPlugin),
], {
  components: components,
});

```
