The Placeholder component for the RTE allows for the display of a customizable placeholder text when the editor is empty.
It helps users locate the editor in the application and prompts them to input content.
The feature works similarly to the native DOM Placeholder attribute used by inputs.
It can be configured to display a specific message or prompt for specific types of blocks.

### Demo

To test the Placeholder component, simply focus on an empty paragraph, and the placeholder will be shown.

### Installation

Follow these steps to integrate the Placeholder component into your Plate editor:


1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-ui-placeholder
```

2. Wrap your block components with the Placeholder component:

```tsx
import {
  ELEMENT_H1,
  ELEMENT_PARAGRAPH,
  withPlaceholders,
} from '@udecode/plate';

export const withStyledPlaceHolders = (components: any) =>
  withPlaceholders(components, [
    {
      key: ELEMENT_PARAGRAPH,
      placeholder: 'Type a paragraph',
      hideOnBlur: true,
    },
    {
      key: ELEMENT_H1,
      placeholder: 'Untitled',
      hideOnBlur: false,
    },
  ]);
```

3. Create your plugins using these components:

```tsx
import { createPlateUI } from '@udecode/plate';

const components = withStyledPlaceHolders(createPlateUI());

const plugins = createPlugins([
  // ...otherPlugins,
], {
  components: components,
});
```
