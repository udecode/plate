The cursor overlay is a component that can be used to display controlled cursors or selections over the editor.
It's the strategy used by [slate-yjs](https://github.com/BitPhinix/slate-yjs) to display the remote cursors.

### Demo

The cursor overlay is shown when the user drags over the editor.

import { CursorOverlaySandpack } from "./CursorOverlaySandpack";

<CursorOverlaySandpack />

### Installation

Follow these steps to integrate the cursor overlay plugin into your Plate editor:


1. Install the required packages:

```bash npm2yarn
npm install @udecode/plate
# or
npm install @udecode/plate-ui-cursor
```

2. Import the plugin and add it to your plugin list:

```
import { createBasicElementsPlugin } from '@udecode/plate-basic-elements';
import { createCursorPlugin } from '@udecode/plate-cursor';
import { createNodeIdPlugin } from '@udecode/plate-node-id';

const plugins = createPlugins([
  ...otherPlugins,
  createBasicElementsPlugin(),
  createNodeIdPlugin(),
  createCursorPlugin(),
], {
  components: components,
});
```
