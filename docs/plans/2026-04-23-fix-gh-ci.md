## Findings

- Task: fix GitHub CI for the current checkout.
- Changeset skill is relevant because earlier review found a possible invalid core changeset bump.
- PR #4941 fails only on `Registry / Validate Registry`.
- CI failure is in `pnpm templates:update --local` while adding `editor-ai.json`.
- `shadcn` calls `bun add @platejs/footnote`; npm returns 404 because `@platejs/footnote` is a new workspace package on this branch and is not published yet.
- The existing local workspace package override runs after shadcn installs registry dependencies, so it is too late for newly added registry packages.
- Fix: when a PR base ref is provided, local registry preparation packs changed workspace package dependencies and rewrites registry `dependencies` entries to `name@file:/absolute/tarball.tgz` before shadcn runs.
