# Launch Next Batch

## When To Run

- when [master-roadmap.md](/Users/zbeyens/git/plate-2/docs/slate-v2/master-roadmap.md)
  has a first unfinished tranche
- after the control docs are already synced to the current doctrine

- [overview.md](/Users/zbeyens/git/plate-2/docs/slate-v2/overview.md)
- [master-roadmap.md](/Users/zbeyens/git/plate-2/docs/slate-v2/master-roadmap.md)

## Hard Rule

Do not launch a new batch from vibes.

If the active tranche is the `slate` core redesign, do **not** launch a
support-package batch just because tranche 4 still exists in the file tree.

Confirm first:

1. the owning tranche is explicit
2. the relevant merged corpus doc exists
3. the control docs do not contradict the tranche objective

## If Those Are Not True

Run one of these first:

- [replan-remaining-work.md](/Users/zbeyens/git/plate-2/docs/slate-v2/commands/replan-remaining-work.md)
