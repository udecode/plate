---
title: "Location API"
description: "Location union helpers for validating paths, points, and ranges."
---

The `Location` interface is a union of the ways to refer to a specific location
in a Plite document: paths, points, or ranges. Methods often accept `Location`
instead of requiring only `Path`, `Point`, or `Range`.

## Type

```typescript
type Location = Path | Point | Range;
```

### Check methods

#### `LocationApi.isLocation(value: unknown) => value is Location`

Check if a value implements the `Location` interface.
