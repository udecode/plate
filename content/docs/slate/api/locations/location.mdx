---
title: "Location API"
---

The Location interface is a union of the ways to refer to a specific location in a Slate document: paths, points or ranges. Methods will often accept a Location instead of requiring only a Path, Point or RangeApi.

```typescript
type Location = Path | Point | Range
```

- [Static methods](location#static-methods)
  - [Check methods](location#check-methods)

## Static methods

### Check methods

#### `LocationApi.isLocation(value: unknown) => value is Location`

Check if a value implements the `Location` interface.
