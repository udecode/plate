## Ecosystem Starting Points

Treat ecosystem packages as compatibility research unless their current release explicitly targets this package set.

- [Plate](https://github.com/udecode/plate) is the product-level editor framework built on Slate.
- [`slate-yjs`](https://github.com/BitPhinix/slate-yjs) is useful prior art for Yjs integration patterns.
- [`slate-angular`](https://github.com/worktile/slate-angular) and [`slate-vue3`](https://github.com/Guan-Erjia/slate-vue3) are useful prior art for non-React view layers.
