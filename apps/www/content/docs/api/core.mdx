## `createPlugins`

Creates a new array of plugins by overriding the plugins in the original array.

<APIParameters>
<APIItem name="plugins" type="PlatePlugin[]" optional>
The original array of plugins.
</APIItem>
<APIItem name="options" type="object" optional>
The options to override plugins.

<APISubList>
<APISubListItem parent="options" name="components" type="Record<string, React.FC>" optional>
Components to override by plugin key.
</APISubListItem>
<APISubListItem parent="options" name="overrideByKey" type="Record<PluginKey, Partial<PlatePlugin>>" optional>
Plugins to override by plugin key.
</APISubListItem>
</APISubList>
</APIItem>
</APIParameters>

<APIReturns>
  An array of plugins with overridden components or properties.
</APIReturns>

<APIReturns>
  An object containing the following properties:
  <APIItem name="pressed" type="boolean">
    A boolean indicating whether the nodeType mark is active in the current
    selection.
  </APIItem>
  <APIItem name="nodeType" type="string">
    The type of the node.
  </APIItem>
  <APIItem name="clear" type="string | string[]">
    Type or types of the node to clear.
  </APIItem>
</APIReturns>
