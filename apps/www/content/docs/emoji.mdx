### Features

The emoji plugin is using the combobox plugin(/docs/combobox). The Emoji plugin enables users to effortlessly insert emojis into their text, enhancing the user experience by providing a simple and convenient way to add visual expression to content.

To use the Emoji plugin, click the emoji icon on the toolbar or use the shortcut ":", followed by the emoji name, such as :apple: üçé. Autocomplete suggestions will appear as you type, and you can select the desired emoji from the list.

### Installation

```bash
npm install @udecode/plate-combobox
npm install @udecode/plate-emoji
```

### Usage

```tsx
import { createComboboxPlugin, createEmojiPlugin } from '@udecode/plate';

const plugins = createPlugins([
  // ...otherPlugins,
  createComboboxPlugin(),
  createEmojiPlugin(),
], {
  components: createPlateUI(),
});
```

#### Options

```tsx
interface EmojiPlugin<TData extends EmojiItemData = EmojiItemData> {
  trigger?: string;
  createEmoji?: CreateEmoji<TData>;
  emojiTriggeringController?: IEmojiTriggeringController;
  id?: string;
}
```
