### Features

This plugin enhances your editor by integrating the Excalidraw library, which allows users to create drawings and diagrams effortlessly. These visual elements are stored as block nodes within Plate, making it simple to incorporate them into your content.

### Installation

```bash
npm install @udecode/plate-excalidraw
```

### Usage

```tsx
import { createSelectOnBackspacePlugin } from '@udecode/plate';
import {
  createExcalidrawPlugin,
  ExcalidrawElement,
  ELEMENT_EXCALIDRAW,
} from '@udecode/plate-excalidraw';

const plugins = createPlugins([
  // ...basicNodesPlugins,
  createExcalidrawPlugin({
    component: ExcalidrawElement,
  }),
  createSelectOnBackspacePlugin({
    options: { query: { allow: [ELEMENT_EXCALIDRAW] } },
  }),
]);
```
