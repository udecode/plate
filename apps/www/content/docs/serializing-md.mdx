### Features

- Convert Markdown content to a Slate value. 

<Callout className="my-4">
**Note**: Converting a Slate value to Markdown is not yet supported.
</Callout>

### Installation

```bash
npm install @udecode/plate-serializer-md
```

### Usage

```tsx
import { createDeserializeMdPlugin } from '@udecode/plate';

const plugins = createPlugins(
  [
    // ...otherPlugins,
    createDeserializeMdPlugin(),
  ],
  {
    components: createPlateUI(),
  }
);
```

### Serializer Usage

Use [remark-slate](https://github.com/hanford/remark-slate).
